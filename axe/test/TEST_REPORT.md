# TEST_REPORT

スプレッドシートおよびスプレッドシートからの報告書（GAS）出力確認項目。

## T-REP-01: GASメニュー表示

- 手順
  1. 対象スプレッドシートを開く
  2. Apps Script に `axe/gas/ReportGenerator.gs` を設定
  3. シートを再読み込み
- 期待結果
  - メニュー `報告書` が表示される

## T-REP-02: 対象タブ抽出

- 手順
  1. `報告書 > Google Docs 報告書を生成` を開く
- 期待結果
  - `検査項目番号` ヘッダーを持つタブのみ候補表示される

## T-REP-03: Docs生成

- 手順
  1. タブを選択して生成実行
- 期待結果
  - Google Docs URL が返る
  - ページ単位で「達成基準リスト」が生成される

## T-REP-04: 記号マッピング

- 手順
  1. 出力されたDocsの結果列を確認
- 期待結果
  - 合格: `○`
  - 不合格: `×`
  - 判定不能: `△`
  - 未検証: `ー`

## T-REP-05: 7/8列互換

- 手順
  1. 7列または8列形式の入力シートで生成
- 期待結果
  - エラーなく読み取りできる

## T-REP-06: Drive管理API（安全確認）

- 手順
  1. `GET /axe/api/drive-cleanup`
- 期待結果
  - 使用量とファイル一覧のJSONを返す
- 注意
  - 通常テストで `deleteAll` は実行しない
