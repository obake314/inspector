### Fix 15: スキャン中 PLAYWRIGHT チェックボックス・ラベルの無効化

**問題**: スキャン実行中でも PLAYWRIGHT チェックボックスとラベルが押下可能だった（DEEP SCAN / MULTI SCAN とは異なる挙動）。  
**原因**: `lockScanUI()` の disabled リストに `playScanOpt` / `batchPlayOpt`、ロックリストに `playScanLabel` / `batchPlayLabel` が含まれていなかった。  
**修正**:
- `lockScanUI()`: `playScanOpt` / `batchPlayOpt` を disabled リストに追加、`playScanLabel` / `batchPlayLabel` をロックリストに追加
- `clearScan()` 内 unlock処理: `playScanLabel` / `batchPlayLabel` を unlock リストに追加、`playScanOpt` / `batchPlayOpt` を disabled 解除リストに追加

### Fix 17: 「該当なし」タブのバッジ数が0になる問題修正

**問題**: 「該当なし」タブにカードが表示されているにもかかわらず、タブ横のバッジ数が 0 になるケースがあった。  
**原因**: バッジ数は `computeTotalScore()` の SC 単位集計値（`ts.na`）を使用しており、BASIC が同一 SC を `pass` として持つ場合に `rank[pass]=1 < rank[na]=3` で pass が勝つため `ts.na=0` になる。一方バケット (`buckets.na`) にはカードが残るため不一致が発生。  
**修正**: `badge.textContent = badgeMap[key] ?? items.length` → `badge.textContent = items.length > 0 ? items.length : (badgeMap[key] ?? 0)` に変更。カードが1件以上あれば実数を表示。

### Fix 19: GAS `getReportTabs()` PERMISSION_DENIED 修正

**問題**: GAS から「報告書生成」ダイアログを開くと `読込エラー: ストレージからの読み取り中にサーバーエラーが発生しました。エラーコード: PERMISSION_DENIED` が発生。  
**原因**: スプレッドシート内に DATASOURCE 型（BigQuery 等に接続）や OBJECT 型のシートが存在する場合、`s.getLastRow()` などの読み取り操作で GAS プラットフォームレベルの PERMISSION_DENIED が発生し、try/catch をすり抜けてリクエスト全体が失敗する。  
**修正**:
- `getReportTabs()`: `s.getType() !== SpreadsheetApp.SheetType.GRID` のシートを先行スキップ
- `readPages_()`: 同様に GRID 以外をスキップ、`getDataRange().getDisplayValues()` も try/catch で保護
- `getCoverUrlMap_()`: 同様に GRID 以外をスキップ、`getDataRange().getDisplayValues()` も try/catch で保護
